Exercice

  1. Commencez par remplir ce modèle avec du contenu
  2. Découpez enssuite ce modèle en 4 fichiers : header, content footer, et index qui inclu les 3
  3. Vous créerez une page php de contenu différente pour chaque rubrique
  4. Navigation : vous écrirez un "contrôleur" dans index.php qui analysera un paramètre de $_GET pour naviguer dans les rubriques.
  5. Contact : Vous ajouterez un formulaire qui post ses données, que vous enregistrerez dans un fichier CSV sur le serveur.
  6. Refactorez la "home" pour aller chercher vos donées dans un fichier, côté serveur.
  7. Ajouter une authentification pour télécharger le fichier des contacts
